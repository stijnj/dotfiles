# Stijn's dotfiles
## Installation
### Using Git and the bootstrap script

Clone this repository in your home folder.

```bash
git clone https://github.com/stijnj/dotfiles.git
cd dotfiles
source install.sh
```

To update, `cd` into your local `dotfiles` repository and then:

```bash
source install.sh
```
